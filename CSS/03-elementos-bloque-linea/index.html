Elementos en línea y en bloque

Los elementos en línea (inline) son las etiquetas HTML que se colocan de forma horizontal una al lado de la otra.

Los elementos en bloque (block) se colocan de forma vertical uno debajo del otro.

Ejemplos:

Las etiquetas <strong></strong> colocan el texto en la misma línea que texto anterior o posterior. También en la misma línea puedo poner un hipervínculo a Google

Las etiquetas <h...></h...> y las etiquetas <p></p> se muestran una debajo de la otra.

Etiquetas div y span

Las etiquetas <div> y <span> son elementos HTML utilizados para controlar la estructura y el estilo de contenido en una página web. Ambas son etiquetas de contenedor sin un propósito semántico específico y se utilizan para agrupar y dar formato a partes específicas de contenido dentro de un documento HTML. Sin embargo, tienen diferentes propósitos y comportamientos.

Las etiquetas tienen el comportamiento predeterminado block y las etiquetas span tienen el comportamiento inline.
