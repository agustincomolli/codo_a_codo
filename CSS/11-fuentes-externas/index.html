Fuentes externas

Para instalar una fuente externa

Buscar la fuente que prefiera en Google Fonts o en otro sitio similar y descargar la familia entera.

Luego copiar el archivo con la tipografía a utilizar en nuestra carpeta fonts:

  • projetc
    • assets
      • fonts
        • jost-regular.woff2
        • jost-regular.ttf

Enlazar la fuente al CSS

En nuestro archivo CSS agregamos el siguiente código:

@font-face {
font-family: "Jost";
src: url("../fonts/jost-regular.ttf") format("truetype");
}

Y luego en el lugar donde se quiera usar la fuente poner por ejemplo:

body {
font-family: "Jost",Arial, Helvetica, sans-serif;
}

Optimizar la fuente

Para reducir el peso en kb de la fuente se puede convertir al formato woff2 a través del sitio: https://cloudconvert.com/
