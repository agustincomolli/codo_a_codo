Formato de texto en HTML

Usando las etiquetas semánticas

En HTML se puede dar formato de cursiva, negrita o subrayado usando las etiquetas de <b>, <i> o <s> pero no es recomendable ya que para eso están las hojas de estilo en cascada CSS.

En caso de necesitar resaltar el texto es conveniente usar las etiquetas semánticas.

Etiquetas semánticas

Este texto está en negrita usando las etiquetas <strong>

Este texto en cambio está en cursiva usando la etiqueta <em>

Puedes usar texto superíndice para escribir por ejemplo: 1ero

Puedes usar texto subíndice para escribir por ejemplo: H2O
